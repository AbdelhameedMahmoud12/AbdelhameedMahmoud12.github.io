Custom ASIC Accelerator

A hardware accelerator for a custom algorithm, designed for low power and high performance.

Project Overview

This project involved the design and implementation of a custom ASIC to accelerate a proprietary signal processing algorithm. The entire RTL-to-GDSII flow was utilized to take the design from a SystemVerilog description to a final layout ready for tapeout. The primary goals were to achieve a 10x performance increase over a software implementation while staying within a strict power budget.

Key Features

  • Highly parallel architecture
  • Custom data path and memory subsystem
  • Power gating and clock gating for low power
  • AMBA AXI4-Lite interface for control

Design & Verification

The design was implemented in SystemVerilog. Functional verification was performed using UVM (Universal Verification Methodology) to ensure comprehensive testing. The design was synthesized using Synopsys Design Compiler and physically implemented using Cadence Innovus. Static Timing Analysis (STA) and power analysis were performed to ensure the design met its performance and power targets.

Tools & Technologies

SystemVerilog UVM Synopsys Tools Cadence Tools ASIC Flow IC Design
