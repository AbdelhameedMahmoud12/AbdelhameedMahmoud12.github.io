CRC Generator using LFSR

A Verilog project that computes 8-bit CRC using a Linear Feedback Shift Register (LFSR). Includes automated testbench verification with file-based inputs and expected outputs.

Project Overview

The design implements an 8-bit Cyclic Redundancy Check (CRC) generator using a Linear Feedback Shift Register (LFSR). The LFSR is initialized with 8'hD8 on reset and processes input data LSB-first. Once all data bits are shifted in, the register holds the CRC value, which is then shifted out serially.

Design Details

  • Seed value: 8'hD8 (asynchronous active-low reset).
  • Supports input length from 1 to 4 bytes (typical: 1 byte).
  • ACTIVE signal enables data shifting, VALID goes high during CRC output.
  • CRC is shifted out bit-by-bit starting from the LSB.

Verification & Testbench

Verification was automated using a testbench (LFSR_CRC_tb.v) that performs:

  1. Reading input bytes from DATA_h.txt (10 test cases).
  2. Reading expected CRC outputs from CRC_OUT_h.txt.
  3. Feeding data to the DUT bit-by-bit and capturing CRC results.

Applications
