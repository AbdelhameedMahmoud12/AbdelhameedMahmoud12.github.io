RISC-V CPU Design

A 5-stage pipelined processor supporting the RV32I instruction set.

Project Overview

This project involved the complete design, implementation, and verification of a 5-stage pipelined RISC-V processor core. The goal was to create a functional and efficient CPU that correctly implements the base integer instruction set (RV32I). The design was written in Verilog and simulated extensively to ensure correctness.

Key Features

  • RV32I Base Integer Instruction Set
  • 5-Stage Pipeline: Fetch, Decode, Execute, Memory, Writeback
  • Full Hazard Detection & Forwarding Unit
  • Static Branch Prediction (Always Not Taken)

Design & Verification

The processor was designed using a modular approach, with separate modules for each pipeline stage, the register file, and control logic. Verification was a critical part of the project. A comprehensive testbench was developed in SystemVerilog, which executed a suite of assembly programs to test all instructions, data hazards, and control hazards. Waveform analysis in QuestaSim was used for debugging.
