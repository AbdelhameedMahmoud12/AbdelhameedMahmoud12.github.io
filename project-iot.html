IoT Weather Station

A web-connected weather station using an ESP32 microcontroller.

Project Overview

This project is a complete IoT weather station that measures temperature, humidity, and pressure, and sends the data to a cloud platform for remote monitoring. The system is built around an ESP32 microcontroller, which handles sensor interfacing and Wi-Fi connectivity.

Key Features

  • Temperature, Humidity & Pressure Sensing
  • Wi-Fi Data Transmission via MQTT
  • Cloud Data Visualization Dashboard
  • Low-Power Deep Sleep Mode for Battery Operation

System Architecture

The device uses a BME280 sensor to collect environmental data. The ESP32, programmed using the Arduino framework and C++, reads the sensor data periodically. It then connects to a local Wi-Fi network and publishes the data to a public MQTT broker. A remote dashboard, subscribed to the MQTT topic, visualizes the incoming data in real-time.

Tools & Technologies

C++ Arduino ESP32 MQTT IoT Embedded
