FPGA Image Processing

Real-time video processing using a Xilinx FPGA.

Project Overview

This project demonstrates real-time image processing on an FPGA. The system captures a video stream, applies a series of filters and transformations in hardware, and outputs the modified video stream. The entire processing pipeline was designed in VHDL and implemented on a Xilinx development board.

Key Features

  • Real-time Sobel Edge Detection
  • Adjustable Color Space Conversion (RGB to Grayscale)
  • VGA Video Input/Output Controller
  • High-throughput, low-latency pipeline

Design & Verification

The image processing algorithms were implemented as a streaming pipeline in VHDL to maximize throughput. Each filter was designed as a separate module. Verification was performed using a simulation testbench that fed the design with raw image files and compared the output against a golden reference generated by a MATLAB model.

Tools & Technologies

VHDL Xilinx Vivado ModelSim FPGA Hardware Acceleration
